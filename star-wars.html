Star Wars DOM manipulation

Using a Star Wars API endpoint named SWAPI (https://swapi.dev/), I built this Javascript logic from scratch to fetch and display this planet/character data in the DOM.

This involves functions that fetch the information and convert it to json, Javascript commands that correctly place the data in the DOM in the appropiate locations and commands that clear previous data.

This was a project for my masters in web developement and I was responsible for all of the Javascript functionality. However, the HTML and styling were set up by the administration.
